# BeezigForge
Forge expansion mod for [Beezig](https://github.com/Beezig/Beezig), the Hive plugin for 5zig.

## Features
+ Settings GUI
+ TIMV enderchests count
+ Point Tags
+ Bedwars teams compass
+ TIMV enderchests compass
+ ...more!
